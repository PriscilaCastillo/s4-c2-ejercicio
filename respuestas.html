Que es posicionamiento

Es el modulo de CSS que define como posicionar los elementos

Explicar los tipos de posicionamiento con ejemplo

Relative, absolute, fixed, sticky y z-index

Explicar las propiedades que se aprende con posicionamiento

Relative: el elemento mantiene su posicion relativa al flujo normal

absolute: El elemento se posiciona respecto a su contenedor

fixed: el elemento queda inamovible en la pantalla

sticky: se mantiene estatico en la pantalla

z-index: se asigna un indice numerico a los elementos para jerarquizar su posicionamiento en la pantalla
